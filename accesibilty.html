El atributo OUTLINE

Importancia del atributo outline en la accesibilidad:

El atributo outline en CSS se ultiza para definir el contorno de un elemento

Cuando se aplica a elementos que pueden recibir el foco (como enlaces, botones, campos de entrada, etc.), el outline

puede ayudar a los usuarios que navegan con el teclado a indentificar qué elemento está actualmente seleccionado.

Para los usuarios que navegan con el teclado, es crucial que puedan ver claramente qué elemento está

enfoncado. El outline proporciona una indicación visual que ayuda a estos usuarios a entender dónde

están en la página y qué acción pueden realizar.

Dentro de la hoja de estilo:

/*Estilo básico para los elementos*/

a, button, input {

margin: 10px;

padding: 10px;

font-size: 16px; /*(Recordemos que este es el tamaño natural de la fuente)*/.

}

/*Estilo de outline para los elementos en focus*/

a:focus, button:focus, input:focus {

outline: 2px solid #007BFF;

outline-offset: 2px;

}

/*Estilo adicional para mejorar la visibilidad*/

a:focus {

back-ground-color: #f0f0f0;

}

button:focus {

back-ground-color: #e0e0e0;

}

input:focus {

back-ground-color: #d0d0d0;

}
