2.Combinadores

El combinador une dos elementos y adjudica una jerarquía y reacción en el diseño:

Combinador descendiente:

div p {
color: blue;
}

Combinador hijo directo:

div > p {
font-weight: bold;
}

Combinador adyacente (+):

h1 + p {
color: red
font-style: italic;
}

Combinador general de hermanos (~):

h1 ~ p {
color: blue
text-decoration: underline;
}

3.Pseudo-clases

Las pseudo-clases adjudican posiciones, reacciones y diseños a elementos y sus posiciones relativas:

Pseudo-clase de enlace visitado:

a:visited {
color: purple;
}

Pseudo-clase de enlace al pasar el ratón por encima:

a:hover {
color: red;
text-decoration: underline;
}

Pseudo-clase de primer hijo:

p:first-child {
font-weight: bold;
color: green;
}

Pseudo-clase de último hijo:

p:last-child {
font-style: italic;
color: blue;
}

Pseudo-clase para elementos activos:

button:active {
background-color: yellow;
}

Pseudo-clase para enfoque:

input:focus {
border: 2px solid orange;
outline: none;
}

Pseudo-clase para elementos vacíos:

div:empty {
border: 2px dashed red;
padding: 10px;
text-aling: center;
}

5.Propiedades

5.1 En este apartado se clasifican las propiedades para tipos de "font-family" en las fuentes:

Aplicar una fuente sans-serif genérica:

body {
font-family: Arial, Helvetica, sans-serif;
background-color: #f4f4f4;
padding: 20px;
}

En el <body>, se aplica una fuente sans-serif (Arial, con fallback a Helvetica o cualquier fuente sans-serif genérica).

Aplicar una fuente serif:

h1 {
font-family: "Times New Roman", Georgia, serif;
color: #333;
}

El <H1> utiliza una fuente serif clásica (Times New Roman con fallback a Georgia).

Aplicar una fuente monospace:

p.code {
font-family: "Courier New", Courier, monospace;
background-color: #ddd;
padding: 10px;
border-radius: 5px;
}

Los párrafos con la clase code utilizan una fuente monoespaciada (Courier New).

Aplicar una fuente decorativa con fallbacks:

h2 {
font-family: "Brush Script MT", "Comic Sans MS", cursive;
color: darkblue;
}

En este <h2>, se utiliza una fuente cursiva decorativa (Brush Script MT con fallback a Comic Sans MS o cualquier fuente cursive).

Aplicar una fuente importada de (Google Fonts):

p.roboto {
font-family: 'Roboto', sans-serif;
color: #555;
}

Se importa la fuente Roboto desde Google Fonts, aplicada a los párrafos con la clase roboto.

5.4 En este apartado observamos los cambios que realiza la propiedad "line-height" en las lineas de texto:

Propiedad line-height
line-height con un valor numérico .line-normal {
font-size: 16px;
line-height: 1.5;
color: skyblue;
}		 En esta línea de texto se puede observar una separación de una vez y media de distancia entre renglones, siendo esta 1/2 veces mayor de lo comunmente aplicado.
line-height con una medida específica en píxeles .line-fixed {
font-size: 16 px;
line-height: 32px;
color: pink;
}		 En esta línea de texto se ha aplicado una medida fijada en píxeles, mostrando una separación total de 32px en cualquier pantalla que la observemos.
Al ser 16 px la medida natural, esta separación sería de 2 veces la medida de la línea.
line-height con una medida reducida .line-small {
font-size: 16px;
line-height: 0.8;
color: lightgreen;
}		 En esta línea de texto se está aplicando una separación menor del tamaño predefinido, dejando un espacio muy reducido entre las líneas.
Normalmente no se suele aplicar ya que da la sensación de aglomeración.
Debemos tener siempre cuidado con la distacia entre líneas, especialmente cuando importamos fuentes externas, ya que pueden tener separación añadida a raíz del diseño de la letra en cuestión.
También hay que recordar que la medida numérica actúa a raíz del tamaño de la letra que decidamos. A mayor el tamaño de la letra, mayor el tamaño de los espacios entre líneas.

5.5 En este apartado observamos los formatos que se pueden usar para aplicar color a las lineas de texto:

Aquí vamos a aplicar el color al texto a través de los nombres predefinidos

}

Este párrafo tiene un color de texto predefinido en inglés, además de un color de fondo en el mismo formato.

Aquí vamos a aplicar el color al texto a través del sistema hexadecimal

.color-hexadecimal {
color: #4CAF50;
background-color: #f4f4f4;
}

Este párrafo tiene un color de texto adjudicado en sistema hexagesimal, además de un color de fondo en el mismo formato.

Aquí vamos a aplicar el color al texto a través del formato RGB (Red, Green, Blue)

.color-rgb {
color: rgb(255, 165, 0);

}

Este párrafo tiene un color de texto en formato RGB, además de un color de fondo en el mismo formato.

}

Este párrafo tiene un color de texto con una transparencia nula, además de un color de fondo con transparencia alta.

Aquí vamos a aplicar el color al texto a través del formato HLS (Hue, Saturation, Light)

.color-hsl {

}

Este párrafo tiene un color de texto adjudicado con el sistema HSL, además de un color de fondo en el mismo formato.

Aquí vamos a aplicar el color al texto a través del formato HSLA (Red, Green, Blue, Alpha)

.color-hsla {
color: hsla(300, 100%, 50%, 1);
background-color: hsla(300, 100%, 50%, 0.2);
}

Este párrafo tiene un color de texto en sistema HSLA con una transparencia nula, además de un color de fondo en el mismo formato con transparencia alta.

Márgenes simétricos y asimétricos

En este apartado vamos a introducir los distintos tipos de márgenes aplicables a un texto, contenedor o imagen:

Este texto y estas imágenes tienen un margen simétrico, aplicado en los cuatro lados por igual

Este texto y estas imágenes tienen un margen asimétrico, aplicado en los cuatro por separado

Hay varias formas de aplicar márgenes (margin) y separaciones (padding) en los textos e imágenes:

Principalmente se aplican en el sentido de las agujas del reloj (arriba, derecha, abajo, izquierda):

.ejemplo {
margin: 20%;
}

Este margen aplicaría un 20% de distancia entre bordes en los 4 costados del texto o la imagen

.ejemplo {
margin: 10px 2px 8px 20px;
}

Este margen aplicaría 10 px arriba, 2px derecha, 8 px abajo y 20 px izquierda de distancia entre bordes en los 4 costados del texto o la imagen

.ejemplo {
margin: 50px 0;
}

Este margen aplicaría 50 px de distancia verticalmente al texto o la imagen

.ejemplo {
margin: 0 50px;
}

Este margen aplicaría 50 px de distancia horizontalmente al texto o la imagen
