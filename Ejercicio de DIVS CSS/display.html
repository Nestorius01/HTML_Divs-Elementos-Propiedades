Distintos tipos de display y sus resultados en <Div>, <img> y (<ul>(Lista desorganizada) o <ol>)(Lista organizada)

Ejemplo de Display: flex

Tenemos una lista desorganizada (ul) que contiene 5 elementos y queremos que se mantengan en la misma línea.

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemento 4
  • Elemento 5

Para ello vamos a crear una clase que llamaremos .flex (en este caso):

.flex {

display: flex;

list-style: none;

color: green;

}

Podemos hacer lo mismo con divs e imágenes. Aunque las imágenes mantendrían este valor por defecto.

Ejemplo de Display: block

En este caso vamos a poner 3 <img> imágenes y las dispondremos en bloque.

Para ello vamos a crear una clase llamada .block (en este caso):

.block {

display: block;

margin: 0px;

width: 100px;

}

Ejemplo de Display: grid

En este caso aplicamos una cuadrícula que marcaremos con 3 columnas y 2 líneas.

Además vamos a hacer que la imagen del centro ocupe el espacio de 2 columnas
y vamos a hacer que la segunda linea ocupe el espacio de dos lineas

Para ello creamos una clase que se llame .grid (en este caso):

.grid {

display: grid;

grid-template-columns: 1fr 2fr 1fr;

grid-template-rows: 1fr 2fr;

}

Luego tratamos las imágenes por separado, para lograr separación y tamaño.

.grid img {

width: 100px;

margin: 10px;

}
